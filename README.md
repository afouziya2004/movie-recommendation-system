# Movie Recommendation System (AI)

A content-based movie recommendation system that suggests
similar movies using AI techniques.

## Features
- Content-based recommendations
- Text similarity using TF-IDF
- Cosine similarity scoring
- Lightweight AI logic

## Tech Stack
- Python
- Scikit-learn
- AI / NLP

## How to Run
pip install -r requirements.txt
python movie_recommender.py
